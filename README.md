# 每日笔记归档器（Daily Note Archiver）

一个 Obsidian 插件，自动管理日记 + 右侧边栏实用工具。

## 功能

### 📁 日记归档
1. **自动创建今日日记** — 每次打开 Obsidian 时，若今日日记不存在则自动创建（支持自定义模板）
2. **自动归档旧文件** — 日记文件夹只保留最近 N 个文件，更早的自动移至 `历史记录/` 子文件夹

### 📺 右侧边栏
3. **🕐 实时时钟 + 下班倒计时** — 显示当前时间（时:分:秒），距下班时间倒计时，过点自动切到明天
4. **📅 日历** — 月视图，可切换上/下月，有日记的日期有下划线标记，点击直接打开当日日记
5. **📋 未完成任务清单** — 扫描指定文件夹下所有笔记中的 `- [ ]` 未勾选待办，支持：
   - ☑ 勾选完成（实时写入源文件）
   - 单击任务文本 → 跳转到对应笔记并定位到该行
   - 自动刷新（文件变更时同步更新）

### ⚙️ 完整设置面板
全部选项可在前台配置，**修改实时生效**，无需重启 Obsidian。

## 安装

1. 将整个 `daily-note-archiver` 文件夹放入你 vault 的 `.obsidian/plugins/` 目录
2. 打开 Obsidian → **设置** → **第三方插件** → **关闭安全模式**
3. 在「已安装插件」列表中找到 **每日笔记归档器**，启用即可

## 配置方式

安装并启用后，打开 **设置 → 第三方插件 → 每日笔记归档器**，所有选项即改即生效。

## 目录结构示例

```
库根目录/
├── 日记/
│   └── 每日/
│       ├── 2025-03-27.md   ← 最近 N 个保留在此
│       ├── 2025-03-26.md
│       └── 历史记录/        ← 超出 N 的自动移到这里
│           ├── 2025-03-20.md
│           └── ...
└── .obsidian/
    └── plugins/
        └── daily-note-archiver/
            ├── manifest.json
            ├── main.js
            ├── styles.css
            └── README.md
```

## 设置项说明

### 📁 归档路径

| 设置项 | 类型 | 说明 |
|--------|------|------|
| 日记文件夹路径 | 文本框 | 存放日记的目录，相对于库根目录 |
| 历史记录文件夹路径 | 文本框 | 旧日记移入的目录 |

### 📌 保留规则

| 设置项 | 类型 | 说明 |
|--------|------|------|
| 保留文件数 | 滑块（1-60） | 当前文件夹保留多少个文件 |

### 📅 日期 & 模板

| 设置项 | 类型 | 说明 |
|--------|------|------|
| 日期格式 | 文本框 | moment.js 格式（`YYYY-MM-DD` / `YYYYMMDD` 等） |
| 标题行 | 文本框 | 文件首行，支持 `{{date}}` 占位符 |
| 正文模板 | 多行文本框 | 标题之后的内容，支持 `{{date}}` 占位符 |

### ⚙️ 归档行为

| 设置项 | 类型 | 说明 |
|--------|------|------|
| 自动创建今日日记 | 开关 | 启动时自动创建 |
| 自动归档旧文件 | 开关 | 启动时自动归档 |
| 立即执行一次 | 按钮 | 手动触发归档 |

### 📺 侧边栏

| 设置项 | 类型 | 说明 |
|--------|------|------|
| 下班时间 | 文本框 | HH:mm 格式，用于倒计时 |
| 待办扫描文件夹 | 文本框 | 扫描此文件夹下所有笔记中的未完成任务 |
| 显示时钟 & 倒计时 | 开关 | 是否显示时钟模块 |
| 显示日历 | 开关 | 是否显示日历模块 |
| 显示未完成任务 | 开关 | 是否显示待办列表模块 |

## 手动触发

打开设置面板点击「**立即执行**」按钮，或在控制台执行：

```js
app.plugins.plugins["daily-note-archiver"].runNow()
```

## 从旧版迁移

如果你之前直接修改过 `main.js` 中的默认值，安装新版后请通过设置面板重新配置——设置面板的修改会存入 `data.json`，优先级高于代码中的默认值。
